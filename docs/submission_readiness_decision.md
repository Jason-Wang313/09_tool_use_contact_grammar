# Submission Readiness Decision

Paper: 09 - Tool Use Contact Grammar
Date: 2026-06-14
Decision: final full-scale current-paper version

## Rationale

The paper now has a clear mechanism, a full manuscript, a deterministic full-scale contact-world experiment, explicit mutation audits, predicate-noise stress, granularity analysis, library-incompleteness analysis, and adversarial counterexamples. The core representational claim is supported inside the controlled symbolic world: ordered contact productions remove false positives that static affordance, unordered contact, local bigram, prior-only, and memory-style representations admit.

The paper is not a complete robot tool-use system. It lacks real-robot evidence, learned contact perception, high-fidelity physics, and continuous controller validation. Those are now explicit future-work boundaries rather than hidden limitations.

## Terminal Recommendation

Use this as the final version for the scoped symbolic representation/falsification claim. For a venue that requires embodied hardware evidence, the next step is a new experiment source rather than more local manuscript hardening.
