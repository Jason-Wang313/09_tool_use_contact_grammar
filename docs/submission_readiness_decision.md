# Submission Readiness Decision

Paper: 09 - Tool Use Contact Grammar
Date: 2026-06-12
Decision: workshop-only / revise

## Rationale

The paper has a clear mechanism, a runnable finite contact-world experiment, explicit ablations, and a new v2 perception-noise stress. The core representational claim is supported inside the controlled world: ordered contact productions remove false positives that static affordance and unordered contact representations admit.

The paper is not ready for a strong main-conference robotics submission because it lacks real-robot evidence, learned contact perception, high-fidelity physics, and continuous controller validation. The v2 stress test is useful because it makes the predicate-perception dependency measurable rather than hidden.

## Terminal Recommendation

Submit only to a workshop or keep revising toward real robot / high-fidelity contact evidence.
